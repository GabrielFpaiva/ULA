# Unidade Lógica Aritmética (ULA)

Este é um programa em Python que simula uma Unidade Lógica Aritmética (ULA). A ULA é um componente crucial em processadores, responsável por realizar operações lógicas e aritméticas em dados.

## Funcionalidades

- Geração de instruções aleatórias de 6 bits.
- Execução de um programa contendo uma sequência de instruções a partir de um arquivo .txt.
- Registro dos resultados de cada instrução em um arquivo de log.

## Arquivos do Projeto

- `ula.py`: Contém a lógica principal do programa.
- `utils.py`: Contém funções utilitárias para geração de instruções aleatórias e execução do programa.
- `program.txt`: Arquivo de entrada contendo a sequência de instruções a serem executadas.
- `log.txt`: Arquivo de saída onde os resultados de cada instrução são registrados.
